O que significa design Mobile-first?

  • A) Um design focado em grandes telas e depois adaptado para pequenos dispositivos.
  • B) Um design focado em pequenos dispositivos e depois adaptado para telas maiores.
  • C) Um design apenas para aplicativos móveis.
  • D) Um design apenas para dispositivos com conexão 5G.

Resposta correta: B

Qual é a principal vantagem do design responsivo?

  • A) Maior tempo de carregamento em dispositivos móveis.
  • B) O site se ajusta automaticamente a diferentes tamanhos de tela.
  • C) Aumento da quantidade de elementos gráficos em telas pequenas.
  • D) Sites que só funcionam em dispositivos móveis.

Resposta correta: B

Qual técnica CSS é usada para criar designs responsivos?

  • A) Media Queries
  • B) Transições CSS
  • C) Flexbox
  • D) Pseudo-classes

Resposta correta: A

Qual a importância de otimizar o design para dispositivos móveis?

  • A) Porque a maioria dos acessos à internet é feita por dispositivos móveis
  • B) Para garantir que o site funcione em desktops antigos
  • C) Para melhorar a performance em telas grandes
  • D) Para facilitar a criação de aplicativos móveis

Resposta correta: A

Qual dessas abordagens é recomendada para começar um projeto responsivo?

  • A) Começar com um layout para desktops e depois adaptar para dispositivos móveis.
  • B) Focar no design para tablets e adaptar para dispositivos maiores.
  • C) Desenvolver para dispositivos móveis primeiro (Mobile-first).
  • D) Criar diferentes versões do site para cada dispositivo.

Resposta correta: C
